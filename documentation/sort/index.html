    Sort

    Sort found objects

    Prefix: sort

    Value: Vector of sort operators

    Can start the pipeline: No

    Steps before: find, get, sort, limit, offset

    Steps after: find, get, sort, limit, offset, update, delete

    Sort operators:

    • Asc - asc(...)
    • Desc - desc(...)
